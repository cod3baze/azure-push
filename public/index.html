Com o nosso bot é possível tirar dúvidas e aprender sobre as linguagems de programação utilizadas nas maiores empresas no ramo da tecnologia.

Linguagens suportadas

  • Python
  • Java
  • Javascript

Exemplos básicos

  • Doctor

  • dr.py: learn

  • dr.jv: contrib

  • Tdr.javascript: community

  • dr.close

Comandos para interagir com o bot:

  • doctor Pega informações sobre o bot
  • dr.linguagem: learn Pega informações de como aprender a linguagem
  • dr.linguagem: contrib Pega informações para contribuir em projetos no github sobre a linguagem.
  • dr.linguagem: community Pega informações das comunidades mais ativas da linguagem
  • dr.close Termina o bot

Recursos usados na contrução do BOT:

  • * NodeJS

    Como Ambiente de execução Javascript

  • * AXIOS

    Fazer requisições asyncronas em API

  • * Github API

    Pegar informações sobre reositórios para contribuições

  • * DISCORD

    Pegar informações sobre Comunidades de TI
